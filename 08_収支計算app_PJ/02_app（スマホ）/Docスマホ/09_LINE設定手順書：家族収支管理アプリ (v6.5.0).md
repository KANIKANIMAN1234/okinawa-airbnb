## 1. 準備物
本作業を開始する前に、以下の情報を手元に用意してください。
* **Vercel デプロイURL:** (例: `https://inoue-family.vercel.app`)
* **管理者LINEアカウント:** チャネル作成および公式アカウント管理に使用します。

---

## 2. LINE Developers での基盤構築
アプリの通信の核となる「チャネル」を作成します。

1. **[LINE Developers Console](https://developers.line.biz/console/)** にログイン。
2. **プロバイダーの作成:** - 「作成」ボタンをクリックし、任意のプロバイダー名（例：井上家プロジェクト）を入力。
3. **チャネルの新規作成:**
   - 「Messaging API」を選択。
   - チャネル名（例：家族収支管理）、チャネル説明、カテゴリを入力。
   - 利用規約に同意して「作成」をクリック。



---

## 4. LIFFアプリの詳細設定
VercelでホスティングしたWebページをLINE内で動かすための設定です。

1. 作成したチャネルの「**LIFF**」タブを選択し、「**追加**」をクリック。
2. **LIFFアプリ名:** 家族収支管理
3. **サイズ:** `Full` (画面全体に表示される設定)。
4. **エンドポイントURL:** VercelのURLを入力。
5. **Scope (権限):** `profile` と `openid` に必ずチェックを入れる（ユーザー名取得に必須）。
6. **ボット情報共有:** `On` に設定。
7. **保存:** 作成後に発行される **LIFF ID**（例: `2009115464-7gzud84W`）をコピー。



---

## 5. アプリコードへのID反映
取得したLIFF IDを、実際のプログラムコードに組み込みます。

1. `index.html` のソースコードを開く。
2. **81行目付近**の定数を書き換える。
   ```javascript
   const LIFF_ID = "ここに取得したLIFF IDを貼り付け";
   
```

## 6. LINE公式アカウントマネージャーの設定
ユーザーが操作するインターフェース（リッチメニュー）を作成し、LIFFアプリをトーク画面に固定します。

1. **[LINE Official Account Manager](https://manager.line.biz/)** にログイン。
2. **応答設定の変更:**
   - [ホーム] ＞ [設定] ＞ [応答設定] を選択。
   - 「応答モード」を **チャット** に設定。
   - 「あいさつメッセージ」は任意ですが、不要であればオフにします。
3. **リッチメニューの新規作成:**
   - [ホーム] ＞ [トークルーム管理] ＞ [リッチメニュー] ＞ [作成] をクリック。
   - **表示期間:** 2026年〜長期（例：2030年までなど）に設定。
   - **メニューバーのテキスト:** 「タップして家計簿を開く」など。
4. **コンテンツ設定（ボタンの割り当て）:**
   - テンプレートを選択（ボタン1つの大きいタイプなどがおすすめ）。
   - **アクション設定:**
     - タイプ: `リンク`
     - URL: **LIFF URL**（例: `https://liff.line.me/2009115464-7gzud84W`）
     - アクションラベル: 「アプリ起動」
5. **保存:** リッチメニューを保存し、ステータスが「有効」になっていることを確認。



---

## 7. 最終動作確認チェックリスト
設定完了後、実機（スマートフォン）のLINEアプリで以下の項目をすべて確認してください。

- [ ] **友だち追加:** 公式アカウントを友だち追加し、トーク画面にリッチメニューが表示されるか？
- [ ] **LIFF起動:** メニューをタップして、LINE内ブラウザでアプリが立ち上がるか？
- [ ] **初回認証:** 権限許可画面（Profile/OpenID）で「許可」を押し、入力画面に遷移するか？
- [ ] **プロファイル表示:** 画面下のログエリアに「Login: [自分のLINE名]」と表示されているか？
- [ ] **公開設定の変更:** LINE DevelopersのLIFF設定ページ上部で、ステータスを `Developing` から `Published`（公開）に変更したか？（これをしないと家族がアクセスできません）



---

## 8. トラブルシューティング（困ったときは）

| 現象 | 主な原因と対策 |
| :--- | :--- |
| **画面が「認証中...」で止まる** | `index.html` に記述した `LIFF_ID` が間違っています。Developers ConsoleのIDと1文字ずつ照合してください。 |
| **「404 Not Found」が出る** | LIFF設定の「エンドポイントURL」がVercelの正しいURL（https://〜）になっているか確認してください。 |
| **家族がアプリを開けない** | LINE Developersの設定が `Developing` のままです。`Published` に変更してください。 |
| **リッチメニューが出ない** | 管理画面の「表示期間」が今日の日付を含んでいるか、ステータスが「有効」になっているか確認してください。 |
| **ログに「Fetch Error」が出る** | GASのウェブアプリURLが古い、またはGAS側のデプロイ設定が「全員（Anyone）」になっていない可能性があります。 |
