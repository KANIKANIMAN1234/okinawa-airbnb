# 役割
あなたはプロのプレゼンテーションデザイナーです。
講座スライドを作成してください。


# 目的
このスライドで、受講者に「この講座は自分にぴったりだ！面白そう！」と感じてもらい、安心して学び続けられるようにします。
講座の世界観を伝え、得られる成果を明確にしたうえで、最終的に自分の商品が作れる意欲が湧くことがゴールです。


# ターゲット
- 副業に興味があるが「売る商品がない」「何を出品すればいいかわからない」と悩む30〜50代。
- AIをこれから始めたい、または始めたばかりの方。
- パソコンやAIの操作に自信がない方も含まれます。


# デザイン・ルール
講座導入の雰囲気が伝わるように、やさしくワクワクする構成にしてください。
- コンセプト: AIの持つ「知的さ・信頼感」と、商品開発の「楽しさ・ワクワク感」を両立させる。
- アスペクト比:16:9
- 配色:
  ・ベースカラー（背景）: `#FFF9EE`（目に優しいオフホワイト）
  ・メインカラー（見出し・図形）: `#0B2A4B`（信頼感のあるディープネイビー）
  ・アクセントカラー（強調・アイコン）: `#FF6F00`（行動を促すエネルギッシュオレンジ）
- 背景:ベースカラーで、シンプルな無地とする。
- 文字:
  ・フォント: `Noto Sans JP` を使用する。
  ・フォントサイズ: 大きめ
  ・ウェイト: 見出しは `Black` で力強く。本文は `Medium` など読みやすい太さで。
  ・基本文字色: `#495057`（可読性の高いダークグレー）
  ・強調文字色: `#E63946`（各ページでのメッセージ内の「月収は30倍以上」など、特に注目させたい箇所に使用。過剰履きの場合は１行に１箇所は強調文字にする）
- レイアウトとビジュアル:
  ・1スライド1テーマを厳守し、情報を詰め込まない。
  ・余白を十分に取り、洗練されたプロフェッショナルな印象に。
  ・各スライドに、内容に合ったシンプルなイラストやアイコンを必ず1つ以上使用する。
    親しみやすさを加えるため、線画やフラットデザインのものが望ましい。
- 言葉遣い:
  ・専門用語を避け、PCやAIに不慣れな方にも理解できるよう、やさしく語りかける口調で作成する。
- CSSにアニメーション（hover, transition, transform など）を含めないでください。
- 動きのあるインタラクションは不要です。すべての要素は静的にレイアウトしてください。
-「**台本**」ブロックは、文脈の確認のみに使用し、はスライド化の対象外です。
-イラストはWeb上にあるアイコンやイラストを利用すること。ChatGPTのDALL-E3を使った画像生成は使用禁止。
-スライドのレイアウトは、以下のサイトのポイントを抑えて行うこと
https://enpreth.jp/media/powerpoint-layout/

# スライド枚数
後述の「構成」のスライドの枚数を厳守し、AIの独自判断による枚数変更はしないでください。
