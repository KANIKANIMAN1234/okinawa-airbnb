# スプレッドシート用 CSV（1ファイル）

`spreadsheet_import.csv` を Google スプレッドシートにインポートして、アプリ用の1つのスプレッドシートを用意する手順です。

## 含まれるシート

この CSV には次の5シート分の**ヘッダー行と初期データ**が入っています（1列目「シート名」で区切り）。

| シート名 | 内容 |
|----------|------|
| **inquiries** | 問い合わせ履歴（受信日時、LINE User ID、表示名、メッセージ本文、メッセージID、管理者通知済） |
| **reservations** | 予約（予約ID、ゲストLINE User ID、チェックイン日、チェックアウト日、宿泊人数、宿泊料金、清掃代、ステータス、確定日時、登録日時、備考） |
| **guests** | ゲストマスタ（LINE User ID、表示名、登録日） |
| **templates** | 通知テンプレート（テンプレートキー、本文、更新日時）＋初期5件 |
| **config** | 設定（設定キー、設定値）＋初期9件 |

## インポート手順（1つのスプレッドシートに5シート作る場合）

1. **新規スプレッドシートを作成**  
   [Google スプレッドシート](https://sheets.google.com)で「空白」の新規作成。

2. **CSV をインポート**  
   - メニュー **ファイル** → **インポート** → **アップロード** で `spreadsheet_import.csv` を選択。  
   - 「インポート場所」で **新しいシートを挿入** を選び、区切り文字は **カンマ** のまま **データをインポート**。

3. **1枚のシートに全データが入った状態になる**  
   1列目が「シート名」、2列目以降が各シートの列です。

4. **5枚のシートに分割する**  
   - シートの下の **+** で新しいシートを4枚追加し、タブ名を次のとおりにする。  
     `inquiries` / `reservations` / `guests` / `templates` / `config`
   - 元のシート（CSV をインポートしたシート）で **データ** → **フィルタを作成** する。
   - 「シート名」列のフィルタで **inquiries** だけ表示 → 表示された行の **2列目以降** をコピー → シート「inquiries」の1行目に貼り付け（1行目がヘッダーになる）。
   - 同様に **reservations** / **guests** / **templates** / **config** をフィルタし、それぞれのシートに2列目以降をコピーする。
   - 分割が終わったら、CSV をインポートした元のシートは削除してよい。

5. **config の値を編集**  
   - シート「config」の **設定値** 列で、次の項目を実際の値に書き換える。  
     - `adminLineUserId` … 管理者の LINE User ID  
     - `airbnbIcalUrl` … Airbnb の iCal カレンダー URL（任意）  
     - その他、URL やフォルダ ID など必要に応じて編集。

これで、GAS から参照する「1つのスプレッドシート」に、inquiries / reservations / guests / templates / config の5シートが用意できます。スプレッドシートの ID を GAS の `SPREADSHEET_ID` に設定してください。
