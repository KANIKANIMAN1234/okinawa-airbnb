# 管理者機能・月次集計仕様書 (v1.0)

## 1. 概要
本ドキュメントは、(1) **管理者の LINE でアクセスした時のみ** 利用できる Vercel 上の設定項目と、(2) **各月が締まったタイミングで月間売上を集計し PDF で Google Drive に保存する** 機能、および (3) **予約確定時の清掃員 LINE グループ／オープンチャットへの自動通知** をまとめる。

---

## 2. 管理者判定と画面の出し分け

### 2.1 判定方法
* Vercel の LIFF では、ページ表示時に `liff.getProfile()` で **LINE User ID** を取得する。
* この ID を GAS の **isAdmin** API に送信する（`action: "isAdmin", data: { lineUserId: "Uxxxx" }`）。
* GAS は config の `adminLineUserId` と照合し、一致すれば `result: { isAdmin: true }` を返す。
* **isAdmin が true の場合のみ**、設定メニュー・設定編集画面・月次PDF用 Drive フォルダID・月締め実行ボタンを表示する。
* **isAdmin が false の場合（宿泊ゲスト）** は、**設定項目は一切表示せず**、カレンダー予約画面のみ表示する。

### 2.2 ゲストがアクセスした場合
* 宿泊ゲストが LINE から同じ LIFF URL にアクセスした場合、管理者用の設定画面・メニューは表示しない。
* カレンダー予約（空き確認・チェックイン/アウト選択・宿泊確定）のみ利用可能とする。

---

## 3. 管理者のみ編集可能な設定項目

以下の項目は、**管理者の LINE ID でアクセスした時のみ** Vercel 画面上で表示・編集できる。編集結果は GAS の saveConfig / saveTemplate API 経由で config または templates に保存する。

| 項目 | 保存先（例） | 説明 |
| :--- | :--- | :--- |
| **Google Drive フォルダID** | config: `driveFolderId` | 月間売上 PDF の保存先。管理者が Drive でフォルダを作成し、URL からフォルダID を取得して入力。**管理者の LINE でアクセスした時のみ設定可**。 |
| 返信メッセージの編集 | templates: `replyMessage` 等 | 問い合わせ等への返信用テンプレート。 |
| 予約確定メッセージ | templates: `reservationConfirm` 等 | ゲストに送る予約確定時の御礼文面。 |
| 一泊単価（価格設定） | config: `nightlyRate` | 数値（円）。 |
| 清掃代（価格設定） | config: `cleaningFee` | 数値（円）。 |
| ペット宿泊ありの場合の清掃代 | config: `cleaningFeeWithPet` | 数値（円）。 |
| 周辺のおすすめレストラン案内 | templates: `restaurantGuide` 等 | 予約確定時にゲストに送る文章。 |
| チェックインガイドPDFのURL | config: `checkInGuidePdfUrl` | チェックイン3日前に送信する URL。 |
| 宿泊1週間前の案内メッセージ | templates: `oneWeekBefore` | 既存。 |
| 玄関ドアの開け方動画のURL | config: `doorVideoYoutubeUrl` | チェックイン当日お昼に送信する YouTube 等の URL。 |
| 清掃員LINEグループID | config: `cleanerLineGroupId` | 予約確定時に通知を送るグループ／オープンチャットのグループID。 |

## 4. 月間売上集計と PDF 保存（Google Drive）

### 4.1 トリガー
* **手動:** 管理者画面に「○年○月を締める」ボタンを配置し、押下時に GAS の「月次集計・PDF生成」API（例: `action: "closeMonth", data: { year: 2025, month: 4, lineUserId: "Uxxxx" }`）を呼び出す。lineUserId が管理者と一致する場合のみ実行する。
* **自動:** 時間駆動トリガーで毎月1日（例: 午前1時）に前月を自動締めし、PDF を生成して Drive に保存する。

### 4.2 集計対象
* reservations シートの **status が「確定」** のレコードのうち、**チェックイン日（または確定日）が対象月に含まれる** ものを集計する。
* 集計内容の例: 売上合計（宿泊料金＋清掃代の合計）、件数、内訳（宿泊料金合計・清掃代合計）、月・年。

### 4.3 PDF の生成と保存
* GAS で集計結果を表形式またはレポート形式の HTML またはテキストにし、PDF 化する（GAS の HTML サービスや Blob を PDF に変換する方法、またはスプレッドシートを PDF 出力する方法を利用）。
* 保存先: **config の `driveFolderId`** で指定した Google Drive のフォルダ。`DriveApp.getFolderById(driveFolderId).createFile(blob)` で保存する。
* ファイル名: 例として「月間売上_YYYY年MM月.pdf」（例: 月間売上_2025年04月.pdf）。
* **driveFolderId が未設定の場合は**、月締め実行をスキップするか、エラーとして管理者に通知する。

### 4.4 フォルダID の設定
* 月次 PDF 用の Google Drive フォルダID は、**管理者の LINE でアクセスした時のみ** Vercel の管理者画面で設定・変更できる。
* ゲストにはこの入力欄を表示しない。

## 5. 予約確定時の清掃員 LINE グループ／オープンチャットへの通知

### 5.1 可否
**可能**。LINE Messaging API の Push Message は、送信先（`to`）に **グループID** を指定できる。

### 5.2 設定手順（運用）
1. 清掃員用の **LINE オープンチャット（またはグループトーク）** を作成する。
2. **公式アカウントのボット** をそのオープンチャット（またはグループ）に **メンバーとして追加** する。
3. そのトークで誰かがメッセージを送る等すると、Webhook に `event.source.type === "group"`, `event.source.groupId` が届く。この **groupId** を控える。
4. 管理者画面（管理者の LINE でアクセスした時のみ表示）で、**清掃員LINEグループID** の入力欄にこのグループID を入力し、保存する（config: `cleanerLineGroupId`）。

### 5.3 予約確定時の処理
* confirmReservation の処理内で、ゲスト・管理者への通知に加え、**config に `cleanerLineGroupId` が設定されていれば**、同じ予約内容の要約（チェックイン日・チェックアウト日・ゲスト名等）を **Push API の `to: cleanerLineGroupId`** で送信する。
* 清掃員向けには「○月○日〜○月○日 予約確定。チェックアウト○月○日。清掃よろしくお願いします。」等の短文でもよい。

### 5.4 注意
* オープンチャットでグループID が取得・送信可能かは LINE の仕様に依存する。通常のグループトークでは利用可能。オープンチャットでもボットを追加したうえでグループID が取得できる場合に本機能を利用する。

---

## 6. API 一覧（管理者・月次関連）

| action | 説明 | 管理者限定 |
| :--- | :--- | :--- |
| **isAdmin** | LIFF の lineUserId が管理者かどうかを返す。 | - |
| **saveConfig** | driveFolderId, nightlyRate, cleaningFee 等を保存。lineUserId が管理者の場合のみ実行可。 | ◯ |
| **getConfig** | 設定値の取得。管理者の場合のみ driveFolderId 等を返す運用も可。 | 取得は管理者推奨 |
| **closeMonth** | 指定年月の月次集計を行い、PDF を生成して Drive に保存。lineUserId が管理者の場合のみ実行可。 | ◯ |

---

## 7. まとめ
* **管理者の LINE でアクセスした時のみ:** Drive フォルダID・返信メッセージ・予約確定メッセージ・一泊単価・清掃代・ペット時清掃代・レストラン案内・チェックインガイドURL・1週間前メッセージ・玄関動画URL・清掃員グループID を Vercel で編集可能。宿泊ゲストがアクセスした場合はこれらは**見えず**、カレンダー予約のみ表示。
* **月が締まったら:** 月間売上を集計し、PDF を管理者が設定した Google Drive のフォルダに保存する。フォルダID は管理者の LINE でアクセスした時のみ設定可能。
* **予約確定時:** ゲスト・管理者に加え、清掃員の LINE グループ／オープンチャットにも自動通知を送ることは**可能**。ボットをグループに追加し、グループID を config に保存して利用する。
