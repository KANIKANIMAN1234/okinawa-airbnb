# 予約禁止期間設定機能

## 概要
管理者が特定の期間を予約禁止に設定できる機能です。メンテナンス、プライベート利用、改装工事などの理由で、一時的に予約を受け付けたくない期間を設定できます。

**重要：** この予約禁止期間は**LINEアプリ上でのみ**ブロックされ、Googleカレンダーには追加されません。つまり、Airbnbには同期されません。

---

## 機能詳細

### 1. 予約禁止期間の設定

**設定画面の場所：**
- LINEアプリ → 設定タブ → 「予約禁止期間設定」セクション

**設定項目：**
- **開始日：** 予約禁止期間の開始日（日付ピッカーで選択）
- **終了日：** 予約禁止期間の終了日（日付ピッカーで選択）
- **理由（オプション）：** 予約禁止の理由（例：メンテナンス、プライベート利用）

**複数期間の設定：**
- 「予約禁止期間を追加」ボタンをクリックして、複数の期間を設定可能
- 例：
  - 4月1日〜4月5日：メンテナンス
  - 5月10日〜5月15日：プライベート利用
  - 6月20日〜6月25日：改装工事

### 2. 予約禁止期間の一覧表示

**表示内容：**
- 開始日 〜 終了日
- 理由（設定している場合）
- 削除ボタン

**デザイン：**
- 黄色の背景（#fff3cd）で目立つ表示
- 各期間ごとに削除ボタンを配置

### 3. 予約禁止期間の削除

**操作：**
1. 一覧から削除したい期間の「削除」ボタンをクリック
2. 確認ダイアログで「OK」をクリック
3. 即座に削除され、カレンダーに反映

---

## カレンダーでの表示

### ブロック表示
- 予約禁止期間の日付は**グレーアウト**（選択不可）
- Airbnb予約、LINE予約と同じように表示される

### ブロック日の種類
1. **Airbnb予約：** Airbnb iCalから取得した予約
2. **LINE予約：** LINEアプリから確定した予約
3. **予約禁止期間：** 管理者が設定した期間

---

## Googleカレンダー・Airbnbとの関係

### 重要な仕様

**予約禁止期間は以下には影響しません：**
- ❌ Googleカレンダーには追加されない
- ❌ Airbnbには同期されない
- ❌ iCal URLには含まれない

**つまり：**
- LINEアプリからの予約のみブロック
- Airbnbからの予約は通常通り受け付けられる
- Googleカレンダーには表示されない

### 使い分け

| 目的 | 使用する機能 |
| :--- | :--- |
| **LINEからの予約のみブロック** | 予約禁止期間設定 |
| **Airbnbからの予約もブロック** | Airbnbの管理画面で手動ブロック、またはGoogleカレンダーに手動で予定を追加 |
| **完全にブロック** | 予約禁止期間設定 + Airbnb手動ブロック + Googleカレンダー追加 |

## データ構造

### スプレッドシート：`blocked_periods`シート

| 列名 | データ型 | 説明 | 例 |
| :--- | :--- | :--- | :--- |
| **periodId** | 文字列 | 期間の一意識別子 | BP1234567890 |
| **startDate** | 日付（文字列） | 開始日（YYYY-MM-DD） | 2026-04-01 |
| **endDate** | 日付（文字列） | 終了日（YYYY-MM-DD） | 2026-04-05 |
| **reason** | 文字列 | 理由（オプション） | メンテナンス |

## API仕様

### getBlockedPeriods
- **権限：** 管理者のみ
- **リクエスト：** `{ "action": "getBlockedPeriods", "data": { "lineUserId": "Uxxxx" } }`
- **レスポンス：** `{ "status": "success", "result": { "periods": [...] } }`

### saveBlockedPeriods
- **権限：** 管理者のみ
- **リクエスト：** `{ "action": "saveBlockedPeriods", "data": { "lineUserId": "Uxxxx", "periods": [...] } }`
- **レスポンス：** `{ "status": "success" }`

## 注意事項

### 1. Airbnbとの同期について
予約禁止期間はLINEアプリ上でのみ有効です。Airbnbからの予約を防ぐには、Airbnbの管理画面で手動でブロックする必要があります。

### 2. 既存予約との関係
予約禁止期間を設定しても、既に確定している予約には影響しません。新規予約のみブロックされます。

### 3. 期間の重複
複数の予約禁止期間が重複していても問題ありません。システムが自動的に統合して処理します。

## 技術詳細

### 日付計算ロジック
予約禁止期間の判定には、年月日を数値化して比較する方式を採用しています。

**例：**
- 2026-12-25 → `20261225`
- 2027-01-06 → `20270106`

```javascript
var currentYMD = year * 10000 + month * 100 + day;
var periodStartYMD = periodStartYear * 10000 + periodStartMonth * 100 + periodStartDay;
var periodEndYMD = periodEndYear * 10000 + periodEndMonth * 100 + periodEndDay;

if (currentYMD >= periodStartYMD && currentYMD <= periodEndYMD) {
  // この日付は予約禁止期間内
}
```

この方式により、年をまたぐ期間（例：2026-12-25〜2027-01-06）でも正確に判定できます。

## トラブルシューティング

### 予約禁止期間が表示されない
1. GASを再デプロイしたか確認
2. スプレッドシートに`blocked_periods`シートが存在するか確認
3. ブラウザのコンソール（F12）でエラーを確認

### カレンダーに反映されない
1. 設定画面で「予約禁止期間を追加」ボタンをクリックしたか確認
2. 「保存しました」メッセージが表示されたか確認
3. 予約タブに戻ってカレンダーを確認

### 月全体がブロックされてしまう
年をまたぐ期間（例：2026-12-25〜2027-01-06）を設定した場合、以前のバージョンでは日付比較ロジックの問題により、開始月全体がブロックされる不具合がありました。最新版では修正済みです。GASを再デプロイしてください。

### Airbnbから予約が入ってしまう
予約禁止期間はLINEアプリ上でのみ有効です。Airbnbからの予約を防ぐには、Airbnbの管理画面で該当期間を手動でブロックしてください。
